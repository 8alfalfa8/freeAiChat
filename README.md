# 🛠️freeAiChat

このリポジトリ（freeAiChat）は、マルチLLM対応AIチャットアプリです。

◆ ユースケース
* コールセンターやサポートデスクの自動化
* 社内ヘルプデスク（IT・人事・総務）の自動応答
* ナレッジベース連携型FAQチャットボット
* 営業・提案支援チャット（CRM連携）

---
